YOU鹅 DeepSeek 全量 AI 更新文件

上传方式：
1. 将本文件夹内的文件按相同目录结构覆盖到 GitHub 仓库根目录。
2. 不需要上传 .env.local，也不要上传任何真实 API Key。
3. Vercel 需要配置环境变量：DEEPSEEK_API_KEY、DEEPSEEK_BASE_URL、DEEPSEEK_MODEL。
4. 上传后在 Vercel 重新部署。

本次包含：
- 服务端 DeepSeek 统一工具 server/deepseek.ts
- 简历解析服务端逻辑 server/resumeAnalyzer.ts
- /api/resume/analyze
- /api/interview/questions
- /api/interview/notify
- /api/interview/feedback
- 前端候选人简历文本解析入口与 AI 状态看板
- DeepSeek 失败时自动 fallback，并提示“当前 DeepSeek 暂不可用，已使用模拟结果。”
